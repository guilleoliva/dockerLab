Dockerlab

Laboratorio de introducción a Docker

View on GitHub Download .zip Download .tar.gz

El objetivo de este laboratorio es obtener los conocimientos básicos sobre Docker.

Los contenidos a tratar son:

  1. Repaso conceptual.Que son los contenedores?

  2. Arquitectura de Docker.

  3. Docker Toolbox. De que se compone?. 3.1 Docker Engine. 3.2 Docker Machine. 3.3 Docker Hub.

  4. Lanzar el primer contenedor.

  5. Docker como herramienta para crear Servicios. 5.1 Creando un microservicio con Node.js

  6. Docker como contenedor de datos. 6.1 Creando un contenedor mongoDB.

  7. Orchestando Servicios con Docker.

  8. Docker Compose.
